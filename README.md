# Financial Avocado

A simple finance managing website built using Symfony framework.

## Opis projektu:
Projekt został stworony z wykorzystaiem Framework'u Symfony 2.8. Dodatkowo wykorzystane zostały:
- FOSUserBundle
- Mysql
- Doctrine
- Bootstrap

### Wymagania serwera:
 - php 5.3.9
 - composer 1.5.6
